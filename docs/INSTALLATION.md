
# 🛠️ دليل التنصيب

## للمطورين
```bash
# 1. استنساخ المستودع
git clone https://github.com/your-username/password-generator.git

# 2. الدخول للمجلد
cd password-generator

# 3. تثبيت المتطلبات
pip install -r backend/requirements.txt

# 4. تشغيل الخادم
python main.py
```


## للمستخدمين العاديين
```bash
# 1. تحميل الصغحة
حمل آخر إصدار من صفحة الإصدارات

# 2. فك الضفط
فك الضغط عن الملف

# 3. فتح الملف
افتح ملف frontend/index.html في متصفحك
```
